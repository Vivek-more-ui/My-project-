# Fitness Application

## Description
This project is a Fitness Application that calculates BMI, calories and nutrition plan using MySQL Stored Procedure.

## Features
- Calculate BMI
- Daily calorie calculation
- Protein, Carbs, Fats calculation
- Store user data

## Technology
- Java (Frontend)
- JDBC (Backend)
- MySQL (Database)

## How it works
1. User enters data
2. System calculates BMI, BMR, TDEE
3. Data is stored in database
4. Result is displayed

## Database
Table: users

## Note
This is a DBMS mini project.
